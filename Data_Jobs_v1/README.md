## Data Jobs Dashboard w/ Power BI

![alt text](/Images/Project1_page1.png)
## Introduction

This project presents an interactive Power BI dashboard built to analyze the Data Job Market. Job market information such as salaries, job demand, and locations is often scattered across different sources. This dashboard brings everything together in a single interactive interface.

Using a dataset of 2024 data job postings, the dashboard helps users explore:

- Job demand across roles

- Salary trends

- Global job distribution

- Market insights for data professionals

This project is useful for **job seekers, career switchers, and data professionals** who want to understand the current data job market trends.

## Dashboard File

**🛠 Tools Used**

- Power BI Desktop

- Power Query Editor

- Data Visualization

- Interactive Dashboard Design

## Skills Demonstrated

**⚙️ Data Transformation (ETL)**

Used Power Query to clean and prepare the dataset:

- Removed unnecessary columns

- Handled missing values

- Converted data types

- Created calculated columns

**🧮 Measures & KPIs**

Created key metrics to analyze the dataset:

- Median Salary

- Average Salary

- Total Job Count

- Job Role Distribution

**📊 Data Visualization**

Different chart types were used to analyze job market trends:

- Column Charts → Compare job demand by role

- Bar Charts → Salary comparison between roles

- Line Charts → Trend analysis over time

- Area Charts → Distribution analysis

**🗺 Geospatial Analysis**

Used Map Visualization to show the global distribution of data jobs and identify regions with higher job demand.

**🔢 KPI Indicators & Tables**

Used Cards and Tables to display important information such as:

- Total job postings

- Median salary

- Top job roles

- Detailed job listings

**🎨 Dashboard Design**

Focused on creating a clean and intuitive dashboard layout so users can quickly understand the insights from the data.

**🖱 Interactive Features Slicers**

Used slicers to allow users to filter the dashboard by job title and other attributes.

**Buttons & Bookmarks**

Implemented buttons and bookmarks to create smooth navigation between dashboard pages.

**Drill-Through**

Enabled drill-through functionality to move from a high-level summary to a detailed view of specific job roles.

## 📊 Dashboard Overview

The dashboard contains two main pages.

### Page 1: Job Market Overview

![alt text](/Images/Project1_page1.png)


This page provides a high-level summary of the data job market, including:

- Total number of job postings

- Median salary

- Most in-demand job titles

- Salary comparison between roles

- Job demand trends

This page helps users quickly understand the overall job market situation.

### Page 2: Job Title Drill-Through Analysis

![alt text](/Images/Project1_page2.png)


This page provides detailed insights for a specific job title, including:

- Salary distribution

- Work-from-home statistics

- Top hiring platforms

- Global job location map

Users can drill through from the main dashboard to analyze detailed information about a specific job role.

## 🎯 Conclusion

This Power BI dashboard demonstrates how raw job posting data can be transformed into meaningful insights using data visualization.By combining data cleaning, visualization, and interactive reporting, the dashboard helps users explore job market trends and make data-driven career decisions.
